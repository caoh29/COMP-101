GIF base quality is not the best, when you scale the image it lossess more quality and when you hover one the opacity changes but its barelly noticiable

JPG base quality is really good, when you scale the image it lossess more quality but not as much as PNG or GIF, the levels of compression are not that noticiable and when you hover over them you can see the effect 100% sure

PNG base quality good, when you scale the image it lossess a lot of quality, the levels of compression are very noticiable and when you hover over them you can see the effect, not as clear as JPEG but more noticiable than GIF

SVG base quality is the best, when you scale the image it remains the same, and when you hover over them you can see the effect similar like a PNG file
